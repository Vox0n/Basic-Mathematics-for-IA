# Отчёт по лабораторной работе №2: Классификация вин с помощью нейронных сетей

## 1. Постановка задачи

**Цель работы:** Разработать и обучить нейронную сеть для классификации вин на три класса на основе их физико-химических показателей.
**Задачи:**

1. Загрузить датасет `load_wine`.
2. Выполнить предобработку данных (One-Hot Encoding целевой переменной).
3. Разделить данные на обучающую, проверочную и тестовую выборки строго по заданному шаблону.
4. Спроектировать нейронную сеть, добиться точности классификации на тестовой выборке **выше 94%**.
5. Зафиксировать структуру сети и количество параметров с помощью метода `.summary()`.

## 2. Реализация (код решения)

```python
import numpy as np
from sklearn.datasets import load_wine
from sklearn.model_selection import train_test_split
from tensorflow.keras import utils
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense
from tensorflow.keras.optimizers import Adam

# 1. Загрузка данных
wine_data = load_wine()
x_data = wine_data.data
y_data = utils.to_categorical(wine_data.target, 3) # One-Hot Encoding

# 2. Разделение данных (строго по шаблону)
x_all, x_test, y_all, y_test = train_test_split(x_data, y_data, test_size=0.1, shuffle=True, random_state=6)
x_train, x_val, y_train, y_val = train_test_split(x_all, y_all, test_size=0.1, shuffle=True, random_state=6)

# 3. Создание архитектуры нейронной сети
model = Sequential([
    Dense(128, input_dim=x_train.shape[1], activation='relu'),
    Dense(64, activation='relu'),
    Dense(3, activation='softmax')
])

# 4. Компиляция и вывод структуры
model.compile(optimizer=Adam(learning_rate=0.001), loss='categorical_crossentropy', metrics=['accuracy'])
model.summary()

# 5. Обучение
history = model.fit(x_train, y_train, epochs=100, batch_size=16, validation_data=(x_val, y_val), verbose=1)

# 6. Оценка качества
results = model.evaluate(x_test, y_test)
print(f"\nТочность на тестовой выборке: {results[1]*100:.2f}%")

```

## 3. Описание архитектуры модели

Для решения задачи была выбрана полносвязная архитектура:

* **Входной слой:** принимает 13 признаков (характеристики вина).
* **Скрытые слои:** 2 слоя по 128 и 64 нейрона с активацией `relu`, что позволяет сети выявлять сложные нелинейные зависимости.
* **Выходной слой:** 3 нейрона (по количеству классов) с функцией `softmax` для получения вероятностного распределения по классам.
* **Оптимизатор:** `Adam` с начальной скоростью обучения 0.001 обеспечивает быструю и стабильную сходимость.
